## v2 综合分公式

```
composite = (PI×2.0 + HP×1.5 + ER + QL + NA + SR + SAT) / 8.5 × 2.0
```

每个维度 0-5 整数分。综合分范围 0-10。

**v1 → v2 关键变化**：
- **公式权重不变**：10 样本验证排序方向正确（同分→同播放，耳鸣/晨僵 4.47→647/626）
- **Bucket 中枢重新校准**：账号初始曝光增长，PI=4 baseline 从 ~200 上调至 ~500-600
- **PI 硬约束新增**：PI 必须是"观看者本人正在经历"的痛点。"替别人焦虑"（如儿科/育儿）在冷启动阶段不计入 PI
- **HP=4 验证注释**：HP=4 的核心价值是降低 2s 跳出率（44.83%），而非提升完播率

## 7 个维度

### PI — Pain Intensity（痛点强度）⭐ 权重 ×2.0
*观众看到这个话题时，是"无所谓随便看看"还是"这说的就是我，我急需答案"？*

- **0** — 观众感觉"无所谓"的日常小事（人人经历但没人当问题）
- **1** — 轻微好奇但不影响生活
- **2** — 有一定困扰但不紧迫
- **3** — 中等困扰，观众会主动搜索
- **4** — 明显影响生活/健康，观众急需答案
- **5** — 观众正在被折磨、反复发作、严重影响日常

> ⚠️ **v2 硬约束**：PI 只计算"视频观看者本人"的痛点。"替家人/孩子/朋友焦虑"在当前冷启动阶段不等效于自身痛点（眨眼 46 播放 vs 同分湿疹 207 播放，差距 4.5x）。儿科/育儿类选题暂时回避。

### HP — Hook Potential（钩子强度）权重 ×1.5
*前 3 秒能不能逼观众看下去？*

- **0** — 通用开场（"大家好..."）
- **3** — 具体的承诺或反直觉断言
- **5** — 一个具体生动的场景，观众无法停止处理

> 📊 **v2 验证注释**：HP=4 样本（早搏）2s跳出率 44.83%（全场最低），HP=3 样本平均 47-50%。HP 主要影响前 2s 留人（决定算法是否继续推荐），而非完播率本身。

### ER — Emotional Resonance（情感共鸣）
*稿子能否让观众产生"这说的就是我"的共鸣？（含健康科普的"症状匹配共鸣"）*

- **0** — 纯信息传递；没有任何共鸣点
- **3** — 一般共鸣；观众能对号入座但不强烈
- **5** — 锐利、具体、让观众忍不住在评论区倾诉自己经历

> 📊 **v2 验证注释**：ER=4 提升完播率（湿疹 18.5% vs 平均 14%）但不直接提升播放量。ER=4+PI=4 组合首次触发真实倾诉评论（早搏）。

### QL — Quotable Lines（金句密度）
*稿子里有没有能被截图/独立传播的句子？*

- **0** — 全是叙述
- **3** — 有一句可记住的
- **5** — 多句独立可用

### NA — Narrativity（叙事性）
*有可辨识的弧线，还是平铺直叙？*

- **0** — 列表式
- **3** — 松散主线
- **5** — 紧凑三幕结构

### SR — Social Resonance（社会议题共振）
*稿子触及当下的社会模式吗？*

- **0** — 纯个人/人际
- **3** — 触到公认现象但没新视角
- **5** — 命名了一个观众认识但没有语言形容的结构性模式

### SAT — Satire Depth（讽刺深度）
*稿子用了多层反讽/戏仿格式吗？*

- **0** — 真诚直陈
- **3** — 一层反讽
- **5** — 嵌套或自指反讽

健康科普走真诚路线，SAT 给 0 即可。

## Bucket 预测（v2 重新校准，基于 10 样本）

| Bucket | 范围 | 含义 |
|---|---|---|
| 底部 | < 100 | PI≤1 或"替别人焦虑"类，算法埋没 |
| 基础盘 | 100 - 1,000 | PI≥4 + 自身痛点的稳定表现区间 |
| 命中 | 1,000 - 10,000 | 第一次破圈 |
| 小爆 | 10,000 - 100,000 | 极罕见的零粉首爆 |
| 大爆 | > 100,000 | 平台算法异常加权 |

**v2 中枢校准（2026-07-24，基于最新 4 条数据）**：
- PI=4 + HP=3 → 中枢 ~500（范围 200-650）
- PI=4 + HP=4 → 中枢 ~550（范围 300-700，样本仅 1）
- PI≤1 或"替别人焦虑" → 中枢 <100

> 注：中枢随账号成长会继续漂移。v1 时期(07-20)中枢~200，v2 时期(07-24)中枢~500，增长 2.5x。下次校准门槛：当 PI=4 样本中枢偏离当前 ±50% 时触发。

## 已验证假设（v2 确认）

1. ✅ **PI 是第一预测变量**：PI≥4 全部 >200 播放，PI≤1 全部 <100（无例外）
2. ✅ **"替别人焦虑"不等效自身痛点**：眨眼(家长焦虑)46 vs 湿疹(自身)207，4.5x 差距
3. ✅ **HP=4 降低 2s 跳出率**：44.83% vs HP=3 平均 47-50%
4. ✅ **ER=4 提升完播率但不直接提升播放量**：湿疹 18.5% 完播但仅 207 播放
5. ✅ **同 composite 同播放**：耳鸣/晨僵均 4.47 → 647/626（1.03x）

## 待验证假设（v2 遗留）

1. **PI=5 的边际效用**：早醒(PI=5)仅 45 播放，但数据可能异常（T+2 且平均播放时长矛盾），待确认
2. **话题搜索热度作为独立变量**：同 PI=4 内部播放量差异 200-650，公式无法捕获，可能需要引入外部热度指标
3. **HP ×1.5 → ×2.0 是否需要**：目前 HP=4 仅 1 样本，证据不足
4. **"倾诉冲动/Comment Trigger"候选维度**：早搏首条倾诉评论支持，但仍仅单样本
