## v1 综合分公式

```
composite = (PI×2.0 + HP×1.5 + ER + QL + NA + SR + SAT) / 8.5 × 2.0
```

每个维度 0-5 整数分。综合分范围 0-10。

**v0 → v1 关键变化**：
- **新增 PI（Pain Intensity 痛点强度）**：最高权重 ×2.0。5 样本一致证据：痛点聚焦度是当前号流量的第一预测变量
- **删除 AB（Audience Breadth 受众广度）**：4 样本证据为反向指标——受众越"普世"完播越差
- **HP 升权 ×1.5**：完播率与播放量强正相关，HP 是文案层面最接近完播预测的维度

## 7 个维度

### PI — Pain Intensity（痛点强度）⭐ 权重 ×2.0
*观众看到这个话题时，是"无所谓随便看看"还是"这说的就是我，我急需答案"？*

- **0** — 观众感觉"无所谓"的日常小事（人人经历但没人当问题）
- **1** — 轻微好奇但不影响生活
- **2** — 有一定困扰但不紧迫
- **3** — 中等困扰，观众会主动搜索
- **4** — 明显影响生活/健康，观众急需答案
- **5** — 观众正在被折磨、反复发作、严重影响日常

### HP — Hook Potential（钩子强度）权重 ×1.5
*前 3 秒能不能逼观众看下去？*

- **0** — 通用开场（"大家好..."）
- **3** — 具体的承诺或反直觉断言
- **5** — 一个具体生动的场景，观众无法停止处理

### ER — Emotional Resonance（情感共鸣）
*稿子能否让观众产生"这说的就是我"的共鸣？（含健康科普的"症状匹配共鸣"）*

- **0** — 纯信息传递；没有任何共鸣点
- **3** — 一般共鸣；观众能对号入座但不强烈
- **5** — 锐利、具体、让观众忍不住在评论区倾诉自己经历

### QL — Quotable Lines（金句密度）
*稿子里有没有能被截图/独立传播的句子？*

- **0** — 全是叙述
- **3** — 有一句可记住的
- **5** — 多句独立可用

### NA — Narrativity（叙事性）
*有可辨识的弧线，还是平铺直叙？*

- **0** — 列表式
- **3** — 松散主线
- **5** — 紧凑三幕结构

### SR — Social Resonance（社会议题共振）
*稿子触及当下的社会模式吗？*

- **0** — 纯个人/人际
- **3** — 触到公认现象但没新视角
- **5** — 命名了一个观众认识但没有语言形容的结构性模式

### SAT — Satire Depth（讽刺深度）
*稿子用了多层反讽/戏仿格式吗？*

- **0** — 真诚直陈
- **3** — 一层反讽
- **5** — 嵌套或自指反讽

健康科普走真诚路线，SAT 给 0 即可。

## Bucket 预测（沿用 v0 方案，待 N≥10 后重校）

当前仍用平台通用默认绝对桶（0 粉新号阶段）：

| Bucket | 范围 | 含义 |

| 底部 | < 100 | 几乎被算法埋了 |

## 待验证假设（v1 遗留）

1. **HP ×1.5 权重是否足够区分完播差异**：v0 时 HP=3 覆盖了跳出率从 40%~53% 的全部样本，维度定义可能需要在未来进一步拆分
2. **SR 在健康科普场景是否有正向贡献**：目前 5 样本里 SR 全部 0-2，无法验证高 SR 的效果
3. **"倾诉冲动/Comment Trigger Intensity"候选维度**：眼红评论区证据支持，但仅单样本，暂不加入公式，继续观察

## 最重要的一句话

**v1 的核心假设：痛点越真、越具体的话题，完播率越高，流量越好。** 写稿时优先选"观众正在被困扰"的话题，而非"人人都相关但没人在乎"的泛科普。
