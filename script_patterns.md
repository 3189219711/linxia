## ⚠️ Pattern 是 toolbox 不是 mold（核心 meta 原则）

任何 pattern 都是**从你已发样本反推的"路径之一"**——不是默认应该 apply 的模板。

如果你每篇都用同一种结构，账号上限只是"做得稳定的中等爆款"。**真正的爆款来自结构创新**。

写稿前**永远先回答**：这篇稿子的最强属性是什么？再决定用什么 pattern。

## 通用 pattern 库（**初始抽象骨架，复盘后填具体**）

### Pattern 1：开头钩子（前 3 秒）
**原则**：绝不用"近年来 / 大家好 / 今天聊一聊"。健康科普尤其要避免"你知道吗"这种通用开场。

### Pattern 2：主体段落数量 + 切分
**原则**：避免 4-7 个并列概念。一般压缩到 **3 个**。

### Pattern 4：emotional 标记节点
**原则**：第一人称"我"出现 0-2 次，永远在 emotional 高潮处。不要空说"我觉得"。

### Pattern 7：句子长度 / 段落节奏（**最容易忽视**）
**写稿用段落版**（每段 100-300 字），不要写一行一行的字幕格式。短句为主，多用语气词。

### Pattern 8：词汇风格
建立你自己的词汇库——这是品牌识别的关键。避免论文体连接词（"其次 / 综上所述 / 因此"）。

### Pattern 9：时长目标
你的 typical_duration_seconds = 90 秒。**不要超过 90 × 1.2 = 108 秒**——越长完播率代价越大。
